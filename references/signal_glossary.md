# 技术分析信号术语表

本文件供 OpenClaw Agent 在生成报告时参考，帮助向用户解释信号含义。

---

## 均线系统 (Moving Averages)

| 信号 | 方向 | 含义 |
|------|------|------|
| MA5/MA10 金叉 | 看多 | 5日均线上穿10日均线，短期趋势走强 |
| MA5/MA10 死叉 | 看空 | 5日均线下穿10日均线，短期趋势走弱 |
| MA5/MA20 金叉 | 看多 | 短期均线上穿中期均线，中短期趋势转多 |
| MA10/MA60 金叉 | 看多 | 中期均线上穿长期均线，中长期趋势确认 |
| 多头排列 | 强看多 | MA5 > MA10 > MA20 > MA60，各周期趋势一致向上 |
| 空头排列 | 强看空 | MA5 < MA10 < MA20 < MA60，各周期趋势一致向下 |
| 站上 MA60 | 看多 | 价格重回60日均线上方，中期趋势改善 |
| 跌破 MA60 | 看空 | 价格跌破60日均线，中期趋势恶化 |

## MACD (指数平滑异同移动平均线)

| 信号 | 方向 | 含义 |
|------|------|------|
| 金叉（零轴上方） | 强看多 | DIF上穿DEA且在零轴上方，强势中的加速上涨 |
| 金叉（零轴下方） | 看多 | DIF上穿DEA但在零轴下方，弱势中可能反弹 |
| 死叉（零轴下方） | 强看空 | DIF下穿DEA且在零轴下方，弱势中继续探底 |
| 死叉（零轴上方） | 看空 | DIF下穿DEA但在零轴上方，上涨动能衰减 |
| 红柱出现 | 看多 | 柱状体由负转正，多方力量开始占优 |
| 绿柱出现 | 看空 | 柱状体由正转负，空方力量开始占优 |
| 顶背离 | 强看空 | 价格创新高但MACD没有跟随创新高，上涨力度衰竭 |
| 底背离 | 强看多 | 价格创新低但MACD没有跟随创新低，下跌力度衰竭 |

## KDJ (随机指标)

| 信号 | 方向 | 含义 |
|------|------|------|
| 金叉（超卖区 K<20） | 强看多 | 低位金叉，反弹概率大 |
| 金叉（中位） | 看多 | 一般性多头信号 |
| 死叉（超买区 K>80） | 强看空 | 高位死叉，回调概率大 |
| 死叉（中位） | 看空 | 一般性空头信号 |
| J > 100 | 看空 | 短期严重超买，注意回调 |
| J < 0 | 看多 | 短期严重超卖，注意反弹 |

## RSI (相对强弱指标)

| 信号 | 方向 | 含义 |
|------|------|------|
| RSI > 80 | 看空 | 超买区域，上涨动能可能透支 |
| RSI < 20 | 看多 | 超卖区域，下跌动能可能衰竭 |
| RSI6/RSI12 金叉 | 看多 | 短期RSI上穿中期RSI，动能转强 |
| RSI6/RSI12 死叉 | 看空 | 短期RSI下穿中期RSI，动能转弱 |

## 布林带 (Bollinger Bands)

| 信号 | 方向 | 含义 |
|------|------|------|
| 突破上轨 | 看空 | 价格超出2倍标准差，短期可能回归均值 |
| 跌破下轨 | 看多 | 价格跌出2倍标准差，短期可能反弹回归 |
| 极度收口 | 中性 | 波动率极低，即将出现方向选择（变盘信号） |

## 成交量

| 信号 | 方向 | 含义 |
|------|------|------|
| 放量上涨 | 看多 | 量价配合，上涨有资金支撑 |
| 放量下跌 | 看空 | 大量抛售，可能加速下跌 |
| 巨量异动（>3倍均量） | 中性 | 成交量极度异常，需结合价格方向综合判断 |
| 极度缩量 | 中性 | 市场参与度极低，观望气氛浓厚 |

## K线形态

| 信号 | 方向 | 含义 |
|------|------|------|
| 十字星 | 中性 | 多空力量均衡，趋势可能反转 |
| 锤子线 | 看多 | 下跌中出现长下影线，下方有强支撑 |
| 射击之星 | 看空 | 上涨中出现长上影线，上方有强压力 |
| 看涨吞没 | 强看多 | 大阳线完全包裹前日阴线，多头强势反转 |
| 看跌吞没 | 强看空 | 大阴线完全包裹前日阳线，空头强势反转 |

## TD Setup 9（九转）

| 信号 | 方向 | 含义 |
|------|------|------|
| TD Buy 9 | 看多 | 连续9根收盘低于4日前收盘，常见于回撤尾段 |
| TD Buy 9（回撤过滤） | 强看多 | 在20日收益不强（<=0）背景下出现，反弹参考价值更高 |
| TD Sell 9 | 看空 | 连续9根收盘高于4日前收盘，常见于短期过热段 |
| TD Sell 9（过热过滤） | 强看空 | 在20日收益较高（>=7%）背景下出现，回落风险提示更强 |
| Perfected Buy/Sell | 增强 | 第8/9根超越第6/7根极值，表示信号更“到位” |

## 信号共振说明

当多个信号同时出现且方向一致时，形成"共振"，信号可靠性显著提高：

- **强看多共振示例**: MACD 零轴上方金叉 + 放量突破 + 均线多头排列
- **强看空共振示例**: MACD 顶背离 + KDJ 超买区死叉 + 跌破 MA60
- **变盘预警**: 布林带极度收口 + 十字星 + 极度缩量 → 方向未定，但爆发在即
